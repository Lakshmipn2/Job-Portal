# Job Portal
## Description
A web application where job seekers can find jobs and employers can post jobs
## Features
- User registration and login
- Role based dashboards (Job Seeker, Employer)
- Employers can post jobs
- Job seekers can browse and apply for jobs
- View applied jobs
## Tech Stack
- Python Flask
- SQLite
- HTML, CSS, Bootstrap
## How to Run
1. Install dependencies: pip install flask flask-sqlalchemy flask-login
2. Run the app: python app.py
3. Open browser: http://127.0.0.1:5000
